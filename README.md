# carrot-py

This is a module to add, subtract, multiply, or divide strings of numbers in Python.

To use, import the module:

`import carrotpy as c`

To add, subtract, multiply, and divide, use:

`c.add("<numbers>")
c.sub("<numbers>")
c.mul("<numbers>")
c.div("<numbers>") `

For example, to add `1` and `2`, use:

`c.add("1 2")`
